Our Contributions and Expertise:

Our team collaboratively contributed to all aspects of the project, bringing a diverse range of skills to the table. We focused on building a robust and scalable application using modern DevOps practices. Our expertise lies in:

  • Containerization with Docker
  • Orchestration with Kubernetes
  • Package Management with Helm
  • CI/CD Automation with GitHub Actions
  • GitOps principles and implementation
  • Continuous Deployment with Argo CD
  • Exposing services using Ingress Controllers

DevOps Tools and Practices Utilized:

We leveraged the following DevOps tools and practices:

  • Dockerfile: For containerizing our application.
  • Kubernetes Manifests: For defining and managing our application deployments.
  • Helm Charts: For packaging and deploying our application on Kubernetes.
  • GitHub Actions: For automating our CI/CD pipelines.
  • GitOps: For managing our infrastructure as code.
  • Argo CD: For continuous deployment and GitOps workflow automation.
  • Ingress Controller: For managing external access to our services.
